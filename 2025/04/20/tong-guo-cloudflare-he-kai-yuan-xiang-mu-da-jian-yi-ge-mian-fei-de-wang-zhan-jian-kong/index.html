本站今日上线了网站监控页面：https://status.undefined 使用的是免费的 Cloudflare Workers 和 lyc8503 大佬的 UptimeFlare 项目，整个流程非常丝滑，五分钟就能搞定。

准备工作

创建存储库

前往项目之后如图所示：

创建了存储库之后可能会遇到一个 Github Action 的报错，这个问题是缺少了 Cloudflare Tokens，下面我们来添加

添加 Cloudflare Tokens

直接前往：https://dash.cloudflare.com/profile/api-tokens 创建即可，选择的时候要选择 Cloudflare Workers 这个选项，需要注意的是要把账户资源和区域资源都改为全部：

设置 Cloudflare Tokens

注意，Tokens 填写一次后将不在可见，请妥善保管

名字的话必须要填写 CLOUDFLARE_API_TOKEN，否则 Github Action 服务不能正常使用：
